## Infos gerais
Há alguns passos para importar o projeto:
1. Instale a unity/unity hub (v6000.2.6f2)
2. Instale o modelo de VR da unity

3. Caso necessário, baixe as pasta "assets" para caso perca os assets do projeto.

OBS:como as pastas do projeto, é muito grande, recomenda-se baixar diretamente a pasta do projeto e descompactá-la. (Mais abaixo).

4. Na unity, vá em  Open -> Selecione a pasta do projeto no file explorer ->  Clique para abrir o projeto

5. Caso haja algum problema, a base do projeto, pode ser baixada direto pelo link:


6. O apk do projeto também esta incluso nesta pasta. Mas é possível construir o projeto usando as opções nativas da unity em: File-> project settings -> build.
